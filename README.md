# Intern-Performance-Data-Pipeline-PostgreSQL-Power-BI-
This project demonstrates an industry-style data pipeline where raw internship performance data is stored in a PostgreSQL database, transformed using SQL views, and visualized using Microsoft Power BI.
This repository presents a complete Data Analytics Pipeline Project designed to analyze intern performance.

The project starts with raw data tables such as interns and tasks, which are stored in PostgreSQL. SQL views are then created to clean and transform the data by removing missing values and preparing it for analysis.

Next, key performance indicators (KPIs) such as:

Average Task Completion Time

Average Project Quality Score

Average Mentor Feedback Score

are calculated using SQL queries.

Finally, Power BI dashboards are built by connecting directly to the analytical SQL view, allowing visualization through KPI cards, bar charts, column charts, and pie charts.

The project demonstrates how modern companies structure data pipelines to separate data storage, transformation, and visualization layers.
